自然数的连续异或和（XOR sum）在计算机科学和算法竞赛中是一个非常经典且优雅的数学性质。

如果我们定义 $f(n)$ 为从 $0$ 到 $n$ 的所有自然数的异或和：

$$f(n) = 0 \oplus 1 \oplus 2 \oplus \dots \oplus n$$

这个序列具有一个**周期为 4** 的极强规律。

### 核心性质：周期律

根据 $n$ 除以 4 的余数（即 $n \bmod 4$），$f(n)$ 的值可以直接通过以下公式得出，无需使用循环逐个计算：

- 当 **$n \bmod 4 = 0$** 时，$f(n) = n$
    
- 当 **$n \bmod 4 = 1$** 时，$f(n) = 1$
    
- 当 **$n \bmod 4 = 2$** 时，$f(n) = n + 1$
    
- 当 **$n \bmod 4 = 3$** 时，$f(n) = 0$


### 原理解析：为什么会这样？

要理解这个规律，我们需要从二进制和异或运算的本质出发。异或运算（$\oplus$）的规则是：**相同为 0，不同为 1**。

#### 1. 相邻偶数与奇数的异或

取任意一个非负整数 $k$，那么 $2k$ 一定是一个偶数，$2k+1$ 是紧随其后的奇数。

在二进制表示中，偶数 $2k$ 的最低位永远是 `0`，而 $2k+1$ 除了最低位变成了 `1`，其他高位与 $2k$ 完全一致。

因此，将它们进行异或：

$$(2k) \oplus (2k+1) = 1$$

_(例如：$4 \oplus 5 = 100_2 \oplus 101_2 = 001_2 = 1$)_

#### 2. 每 4 个数的一组抵消

基于上述结论，如果我们取连续的 4 个数，从 4 的倍数开始（即 $4k, 4k+1, 4k+2, 4k+3$），我们可以将它们两两分组：

$$[4k \oplus (4k+1)] \oplus [(4k+2) \oplus (4k+3)]$$

根据前面的结论，这等于：

$$1 \oplus 1 = 0$$

这意味着，**任何从 0 开始、到 $4k+3$ 结束的连续自然数序列，其异或和必然为 0**。

#### 3. 推导 4 种情况

基于“每 4 个数异或和为 0”的结论，我们可以轻松推导出前面的 4 种情况：

- **当 $n = 4k$ 时**（如 0, 4, 8）：
    
    $f(n) = f(4k-1) \oplus 4k$
    
    因为 $4k-1$ 属于 $4k+3$ 的形式（前一个周期的结尾），所以 $f(4k-1) = 0$。
    
    因此，$f(n) = 0 \oplus n = n$。
    
- **当 $n = 4k+1$ 时**（如 1, 5, 9）：
    
    $f(n) = f(4k) \oplus (4k+1)$
    
    代入上一步的结果：$4k \oplus (4k+1) = 1$。
    
    因此，$f(n) = 1$。
    
- **当 $n = 4k+2$ 时**（如 2, 6, 10）：
    
    $f(n) = f(4k+1) \oplus (4k+2)$
    
    代入上一步的结果：$1 \oplus (4k+2)$。
    
    因为 $4k+2$ 是偶数，最低位是 0，与 1 异或就相当于加 1，结果是 $4k+3$。
    
    因此，$f(n) = n + 1$。
    
- **当 $n = 4k+3$ 时**（如 3, 7, 11）：
    
    这就是完整的一个周期，所有数刚好两两抵消。
    
    因此，$f(n) = 0$。
    

### 实际应用：求任意区间 $[L, R]$ 的异或和

利用 $f(n)$ 的性质，我们可以将求异或和的时间复杂度从 $O(N)$ 降到 $O(1)$。

如果我们想求从 $L$ 到 $R$ 的所有自然数的异或和：

$$L \oplus (L+1) \oplus \dots \oplus R$$

利用异或运算“**一个数异或自己等于 0**”的性质（即 $A \oplus A = 0$），我们可以将问题转化为前缀异或和的差分：

$$[L, R]\text{ 的异或和} = f(R) \oplus f(L-1)$$

你只需要用上面的周期律分别在 $O(1)$ 时间内算出 $f(R)$ 和 $f(L-1)$，然后将它们异或即可得出最终结果。

## 推广形式：
### 一、 步长为 $x$（等差数列 $0, x, 2x, \dots, nx$）

如果序列是由连续加 $x$ 产生的，规律会根据 $x$ 的特征发生分化：

#### 1. 当 $x$ 是 2 的幂时（$x = 2^k$）：完美继承周期律

根据 $n/x$ 除以 4 的余数（即 $(n+x) \bmod 4$），$f(n)$ 的值可以直接通过以下公式得出，无需使用循环逐个计算：

- 当 **$(n/x) \bmod 4 = 0$** 时，$f(n) = n$
    
- 当 **$(n/x) \bmod 4 = 1$** 时，$f(n) = x$
    
- 当 **$(n/x) \bmod 4 = 2$** 时，$f(n) = n + x$
    
- 当 **$(n/x) \bmod 4 = 3$** 时，$f(n) = 0$

#### 二、 当 $x$ 是任意正整数时：周期破坏，需用“拆位法”

如果 $x$ 不是 2 的幂（比如 $x = 3$，序列为 $0, 3, 6, 9 \dots$），由于加法产生的进位会混合二进制的高低位，前面提到的简单周期规律就被打破了。

面对这种任意步长的等差数列，通常没有 $O(1)$ 的公式。标准的解法是**按位统计（拆位法）**：

- 独立观察结果的每一位（第 $i$ 位）。
    
- 统计序列中所有数字在第 $i$ 位上共有多少个 `1`。
    
- 如果 `1` 的总数是奇数，结果的该位就是 `1`；如果是偶数，则是 `0`。
    
    这种方法会将时间复杂度从 $O(1)$ 退化到 $O(\log(\max))$，但依然比直接用循环计算要高效得多。

#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    ll l, r, x;
    cin >> l >> r >> x;
    r = (r / x) * x;
    ll ans = 0;
    if ((r / x) % 4 == 0)
        ans = r;
    else if ((r / x) % 4 == 1)
        ans = x;
    else if ((r / x) % 4 == 2)
        ans = r + x;
    else
        ans = 0;
    l--;
    l = (l / x) * x;
    if ((l / x) % 4 == 0)
        ans ^= l;
    else if ((l / x) % 4 == 1)
        ans ^= x;
    else if ((l / x) % 4 == 2)
        ans ^= l + x;
    else
        ans ^= 0;
    cout << ans << endl;
}
