前置知识：[[叉积]]  [[克莱姆法则（Cramer's Rule）]]
1.快速排斥实验

在进行复杂的数学运算之前，我们可以先通过一个简单的测试来快速排除那些“绝对不可能相交”的线段。

假设我们有线段 $AB$，端点为 $A(x_1, y_1)$ 和 $B(x_2, y_2)$；以及线段 $CD$，端点为 $C(x_3, y_3)$ 和 $D(x_4, y_4)$。

我们可以分别为两条线段构建**最小包围盒**（即以线段为对角线的矩形）。如果这两个矩形在平面上根本没有任何重叠，那么这两条线段必然不可能相交

**判断逻辑：**

如果以下任意一个条件成立，则两个包围盒不相交，线段必然**不相交**：

- 线段 $AB$ 包围盒的右边界 < 线段 $CD$ 包围盒的左边界：$\max(x_1, x_2) < \min(x_3, x_4)$  
- 线段 $CD$ 包围盒的右边界 < 线段 $AB$ 包围盒的左边界：$\max(x_3, x_4) < \min(x_1, x_2)$
- 线段 $AB$ 包围盒的上边界 < 线段 $CD$ 包围盒的下边界：$\max(y_1, y_2) < \min(y_3, y_4)$
- 线段 $CD$ 包围盒的上边界 < 线段 $AB$ 包围盒的下边界：$\max(y_3, y_4) < \min(y_1, y_2)$

如果通过了快速排斥实验，说明两线段“有相交的可能”，接下来进入跨立实验。

2.跨立实验

跨立实验是判断线段相交的核心。如果线段 $AB$ 和 $CD$ 相交，那么线段 $AB$ 的两个端点 $A$ 和 $B$ 必然分布在直线 $CD$ 的两侧；同理，线段 $CD$ 的两个端点 $C$ 和 $D$ 必然分布在直线 $AB$ 的两侧。这被称为**相互跨立**。

我们利用**向量叉积**来判断点在直线的哪一侧。

对于二维向量 $\vec{u} = (x_u, y_u)$ 和 $\vec{v} = (x_v, y_v)$，它们的叉积定义为：

$$
\vec{u} \times \vec{v} = x_u \cdot y_v - x_v \cdot y_u
$$

叉积结果的符号具有重要几何意义：

- 若 $\vec{u} \times \vec{v} > 0$，表示 $\vec{v}$ 在 $\vec{u}$ 的逆时针方向。

- 若 $\vec{u} \times \vec{v} < 0$，表示 $\vec{v}$ 在 $\vec{u}$ 的顺时针方向。

- 若 $\vec{u} \times \vec{v} = 0$，表示两向量共线。

跨立实验的条件：

  1. **判断 $C, D$ 是否跨立 $AB$：**

     计算 $\vec{AB} \times \vec{AC}$ 和 $\vec{AB} \times \vec{AD}$。如果 $C$ 和 $D$ 在直线 $AB$ 两侧，这两个叉积的符号必定相反，即它们的乘积小于或等于 $0$（等于 $0$ 表示端点恰好落在另一条直线上）：

     $$
     (\vec{AB} \times \vec{AC}) \cdot (\vec{AB} \times \vec{AD}) \le 0
     $$

  2. **判断 $A, B$ 是否跨立 $CD$：**

     同理，计算：
 $$
     (\vec{CD} \times \vec{CA}) \cdot (\vec{CD} \times \vec{CB}) \le 0
     $$

  只有当**上述两个条件同时满足时**，线段 $AB$ 和 $CD$ 才真正相交。

3.计算交点坐标

当我们确认两条线段相交后，就可以安全地计算交点了。此时推荐使用**参数方程**，它可以完美避免斜率不存在（垂直线）的问题。

对于线段 $AB$ 上的任意一点 $P$，可以用参数 $t$ 表示：

$$
P = A + t(\vec{AB}) \quad (0 \le t \le 1)
$$

展开为坐标形式：

$$
x = x_1 + t(x_2 - x_1)
$$

$$
y = y_1 + t(y_2 - y_1)
$$

同理，线段 $CD$ 上的任意一点可以用参数 $u$ 表示：

$$
x = x_3 + u(x_4 - x_3)
$$

$$
y = y_3 + u(y_4 - y_3)
$$

当两线段相交时，这两组坐标相等。我们得到一个关于 $t$ 和 $u$ 的二元一次方程组：

$$
x_1 + t(x_2 - x_1) = x_3 + u(x_4 - x_3)
$$

$$
y_1 + t(y_2 - y_1) = y_3 + u(y_4 - y_3)
$$

根据克莱姆法则（Cramer's Rule）求解 $t$。令：

$$
\begin{aligned}
D_x = x_2 - x_1, D_y = y_2 - y_1\\
E_x = x_4 - x_3, E_y = y_4 - y_3\\
F_x = x_3 - x_1, F_y = y_3 - y_1
\end{aligned}
$$

方程组可改写为：

$$
t \cdot D_x - u \cdot E_x = F_x
$$

$$
t \cdot D_y - u \cdot E_y = F_y
$$

求出分母（实际上就是 $\vec{AB}$ 和 $\vec{CD}$ 的叉积）：

$$
denominator = D_x \cdot E_y - D_y \cdot E_x
$$

由于我们前面已经确认线段相交，如果 $denominator \neq 0$（即两条线段不平行或共线重叠），解得：

$$
t = \frac{F_x \cdot E_y - F_y \cdot E_x}{denominator}
$$

最后，将 $t$ 代回 $AB$ 的参数方程即可得到交点坐标 $(X_{intersect}, Y_{intersect})$：

$$
X_{intersect} = x_1 + t(x_2 - x_1)
$$
$$
Y_{intersect} = y_1 + t(y_2 - y_1)
$$

*(注：如果 $denominator = 0$ 且两线段通过了跨立实验，说明它们是共线且重叠的，此时交点不是唯一的，而是线段的一段重叠区间。)*

```cpp
class Solution
{
public:
    //二维点/向量
    struct point
    {
        double x, y;
        point(double _x = 0, double _y = 0) : x(_x), y(_y) {}
        point operator+(const point &b) const
        {
            return point(x + b.x, y + b.y);
        }
        point operator-(const point &b) const
        {
            return point(x - b.x, y - b.y);
        }
        point operator*(double k) const
        {
            return point(x * k, y * k);
        }
    };
    //快速排斥实验
    bool test(point a, point b, point c, point d)
    {
        return !(max(a.x, b.x) < min(c.x, d.x) ||
                 max(c.x, d.x) < min(a.x, b.x) ||
                 max(a.y, b.y) < min(c.y, d.y) ||
                 max(c.y, d.y) < min(a.y, b.y));
    }
    //二维叉积
    double cross(point a, point b)
    {
        return a.x * b.y - b.x * a.y;
    }
    //三点叉积
    double cross(point a, point b, point c)
    {
        return cross(b - a, c - a);
    } 
    // 点c是否在线段ab上
    bool check(point a, point b, point c)
    {
        return min(a.x, b.x) <= c.x &&
               c.x <= max(a.x, b.x) &&
               min(a.y, b.y) <= c.y &&
               c.y <= max(a.y, b.y);
    }
    vector<double> intersection(vector<int> &start1, vector<int> &end1, vector<int> &start2, vector<int> &end2)
    {
        point a, b, c, d;
        a.x = start1[0], a.y = start1[1];
        b.x = end1[0], b.y = end1[1];
        c.x = start2[0], c.y = start2[1];
        d.x = end2[0], d.y = end2[1];
        // 快速排斥实验
        if (!test(a, b, c, d))
        {
            return {};
        }
        // 跨立实验
        if (cross(a, b, c) * cross(a, b, d) > 0 || cross(c, d, a) * cross(c, d, b) > 0)
        {
            return {};
        }
        // 计算坐标
        double dm = cross(a - b, c - d);
        if (dm == 0)
        {
            // 重叠
            // 返回的点必是端点
            vector<point> points;
            if (check(c, d, a))
                points.push_back(a);
            if (check(c, d, b))
                points.push_back(b);
            if (check(a, b, c))
                points.push_back(c);
            if (check(a, b, d))
                points.push_back(d);
            sort(points.begin(), points.end(), [](point &a, point &b){
                if(abs(a.x-b.x)>1e-6)
                    return a.x < b.x;
                return a.y < b.y; });
            return {points[0].x, points[0].y};
        }
        else
        {
            double t = cross(c - a, d - c) / dm;
            return {a.x + t * (b.x - a.x), a.y + t * (b.y - a.y)};
        }
    }
};
```
