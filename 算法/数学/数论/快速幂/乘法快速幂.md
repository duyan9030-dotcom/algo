快速幂（Fast Exponentiation）是计算机科学中一个非常经典且优雅的算法。它的核心目的是在极短的时间内计算出 $a^n$ 的结果（在实际应用中，通常还会伴随取模运算 $a^n \pmod m$，以防止数字溢出）。

如果用最朴素的乘法，计算 $a^n$ 需要将 $a$ 连乘 $n$ 次，时间复杂度是 $O(n)$。当 $n$ 非常大（比如 $10^9$）时，这种方法会非常缓慢。而**快速幂可以将时间复杂度大幅降低到 $O(\log n)$**。

### 核心思想：把指数“拆”成二进制

快速幂利用了**二进制**和**倍增**的思想。我们可以把指数 $n$ 看作是一个二进制数。

举个直观的例子，假设我们要计算 $3^{11}$：

1. 将指数 $11$ 转换为二进制：$11$ 的二进制是 `1011`。
    
2. 这意味着 $11 = 8 + 2 + 1$，或者写成 $11 = 2^3 + 2^1 + 2^0$。
    
3. 根据指数的运算性质，$3^{11}$ 可以被拆解为：
    
    $$3^{11} = 3^{8+2+1} = 3^8 \times 3^2 \times 3^1$$

发现了吗？我们不需要乘 $11$ 次。我们只需要知道 $3^1, 3^2, 3^4, 3^8...$ 的值，然后把需要的项乘起来即可。

而 $3^2$ 是 $3^1$ 的平方，$3^4$ 是 $3^2$ 的平方，$3^8$ 是 $3^4$ 的平方。**每一次底数的平方，都对应着二进制位向左移动一位。**

### 算法步骤

1. 初始化一个变量 `result = 1`（用来保存最终结果），以及 `base = a`（用来不断平方以提供 $a^1, a^2, a^4, a^8...$）。
    
2. 当指数 $n > 0$ 时，进入循环：
    
    - **判断当前最低位**：如果 $n$ 的二进制最低位是 $1$（即 $n \pmod 2 \neq 0$ 或 `n & 1 == 1`），说明我们需要当前这一项，于是将 `result` 乘上当前的 `base`。
        
    - **底数翻倍**：将 `base` 自身平方（`base = base * base`），准备给二进制的下一位使用。
        
    - **指数减半**：将 $n$ 除以 $2$（或右移一位 `n >>= 1`），抹去已经处理过的最低位。
        
3. 循环结束，返回 `result`。
[快速幂模板](https://www.luogu.com.cn/problem/P1226)
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
ll power(ll a,ll b,ll p)
{
	int res = 1;
	while(b>0)
	{
		if(b%2==1)
		{
			res = (res * a) % p;
		}
		a = (a * a) % p;
		b >>= 1;
	}
	return res;
}
int main()
{
	ll a, b, p;
	cin >> a >> b >> p;
	cout << a<<"^"<<b<<" mod "<<p<<"="<<power(a,b,p);
	return 0;
}
```
