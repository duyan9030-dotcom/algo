# 一、埃式筛
埃拉托斯特尼筛法（Sieve of Eratosthenes，简称埃氏筛）是古希腊数学家埃拉托斯特尼提出的一种简单且高效的求解素数的算法。它的核心思想非常直观：**素数的倍数一定不是素数。**

以下是关于埃氏筛从原理、代码实现到时间复杂度推导的详细解析。

### **1、 算法原理**

埃氏筛的操作过程就像是用一个筛子，不断滤掉合数，最后留在筛子里的就只有素数。具体步骤如下：

1. **初始化：** 创建一个长度为 $n+1$ 的布尔数组（或列表），假设所有数最初都是素数（标记为 `true`）。0 和 1 不是素数，直接标记为 `false`。
    
2. **寻找素数：** 从最小的素数 2 开始遍历数组。
    
3. **筛除倍数：** 如果当前数字 $i$ 被标记为素数，那么将它在数组中所有的倍数（$2i, 3i, 4i \dots$）全部标记为合数（`false`）。
    
4. **递推：** 寻找下一个没有被标记为 `false` 的数字，重复步骤 3。
    
5. **终止条件：** 当遍历到 $\sqrt{n}$ 时即可停止。因为如果一个合数 $k$ 包含大于 $\sqrt{n}$ 的质因子，那么它必定也包含一个小于或等于 $\sqrt{n}$ 的质因子，早就已经被筛掉了。

**核心优化点：**

在筛除倍数时，不需要从 $2i$ 开始，而是可以直接从 $i^2$ 开始。因为对于 $i$ 的较小倍数（如 $2i, 3i$），在之前遍历较小素数（2, 3）时，就已经被作为 2 和 3 的倍数筛除掉了。

### **2、 代码实现**

以下是加入上述优化后的标准埃氏筛实现：

C语言实现

```cpp
vector<int> primes;
bool isprime[100000005];
void build1(ll n)
{
	for (ll i = 1; i <= n; i++)
	{
		isprime[i] = true;
	}
	for (ll i = 2; i <= n; i++)
	{
		if (isprime[i])
		{
			primes.push_back(i);
			for (ll j = i * i; j <= n; j += i)
				isprime[j] = false;
		}
	}
}
```

### **3、 时间复杂度计算**

埃氏筛的时间复杂度是 **$O(n \log \log n)$**。这个结论的推导需要借助一些数论知识。

**1. 操作次数的拆解**

外层循环寻找素数，内层循环去划掉该素数的倍数。对于任意一个素数 $p$，我们在范围 $n$ 内需要划掉的倍数个数大约是 $\frac{n}{p}$ 个。

因此，算法的总操作次数 $T(n)$ 实际上就是所有素数倍数个数的加总：

$$T(n) = \sum_{p \le n} \frac{n}{p} = n \sum_{p \le n} \frac{1}{p}$$

（注意：这里只对素数 $p$ 进行求和，如果是对所有整数求和，那就是调和级数，结果是 $O(n \log n)$。但这正是埃氏筛的精妙之处——它只对素数的倍数进行操作）。

**2. 素数倒数和的渐进界**

为了求解 $\sum_{p \le n} \frac{1}{p}$，我们需要用到数论中的**梅滕斯第二定理 (Mertens' 2nd theorem)**。该定理证明了小于等于 $n$ 的所有素数的倒数之和，其渐近行为满足：

$$\sum_{p \le n} \frac{1}{p} \approx \ln(\ln n) + M$$

其中 $M$ 是梅滕斯常数（大约为 0.261497）。

**3. 得出最终复杂度**

将梅滕斯第二定理代回原来的操作次数公式中：

$$T(n) \approx n (\ln(\ln n) + M)$$

去掉低阶项和常数，最终的时间复杂度即为：

$$O(n \log \log n)$$

**空间复杂度：**

由于需要维护一个长度为 $n$ 的布尔数组，空间复杂度为 $O(n)$。在极限优化下，可以通过位图（Bitset）将空间常数压缩至原来的 1/8 或 1/32，或者使用“分段筛”来进一步降低空间复杂度。
# 二、欧拉筛
### 1. 算法原理

欧拉筛（线性筛）的核心目标是解决埃氏筛中**合数被多次重复标记**的问题。它的根本原则只有一条：

**“任何一个合数，只被它最小的质因数筛掉，且仅筛一次。”**

为了实现这个原则，算法维护两个数据结构：

1. 一个布尔数组，用于记录每个数是否为质数（默认全是，然后把合数筛掉）。
    
2. 一个有序列表，按从小到大的顺序记录当前已经找到的所有质数。

**运行逻辑：**

从 $2$ 到 $N$ 遍历每一个数 $i$：

- 如果 $i$ 没有被标记为合数，那它就是质数，将其加入质数列表。
    
- **不管 $i$ 是质数还是合数**，我们都遍历当前质数列表中的质数 $p_j$（从小到大）：
    
    - 将 $i \times p_j$ 标记为合数。
        
    - **关键防线**：如果 $i$ 能够整除 $p_j$（即 `i % pj == 0`），立刻停止内层循环（`break`）。
        

这个 `break` 条件保证了，用来构造合数 $i \times p_j$ 的质数 $p_j$，永远是这个合数的**最小质因数**。

### 2. 代码实现

以下是 C++ 版本的标准实现，逻辑非常干练。

C++

```cpp
vector<int> primes;
bool isprime[100000005];
void build2(ll n)
{
	isprime[0] = isprime[1] = true;
	for(ll i = 2; i <= n; i++)
	{
		if (!isprime[i])
			primes.push_back(i);
		for (ll j = 0; j < primes.size() && i * primes[j] <= n; j++)
		{
			isprime[i * primes[j]] = true;
			if (i % primes[j] == 0)
				break;
		}
	}
}
```

### 3. 复杂度证明

我们需要证明欧拉筛的时间复杂度是严格的 $O(N)$。

要证明这一点，等价于证明：**在 $1$ 到 $N$ 的范围内，内层循环的标记语句 `is_prime[i * primes[j]] = false`，对于每一个合数 $C$，精确地只执行了 $1$ 次。**

**证明过程：**

1. **合数唯一分解定理**：
    
    假设有一个任意合数 $C \le N$。根据算术基本定理，任何合数都可以唯一分解为质数的乘积。
    
    因此，$C$ 必然存在一个**唯一且最小的质因数**，我们记为 $p_{min}$。
    
2. **拆分合数 $C$**：
    
    我们可以把合数 $C$ 写成：$C = p_{min} \times i$ （其中 $i = C / p_{min}$）。
    
    因为 $p_{min}$ 是 $C$ 的最小质因数，这就意味着，**$i$ 的最小质因数必然大于或等于 $p_{min}$**。
    
    _(反证法：如果 $i$ 有一个质因数 $p' < p_{min}$，那么 $C$ 也必能被 $p'$ 整除，这与 $p_{min}$ 是 $C$ 的最小质因数矛盾。)_
    
3. **对应到算法逻辑中**：
    
    外层循环遍历的是 $i$，内层循环遍历的是 $p_j$。
    
    当我们外层循环进行到数字 $i$，内层循环按顺序取出质数 $p_j$ 时，只要 $p_j < \text{最小质因数}(i)$，`i % p_j != 0`，循环会继续，正常筛除 $i \times p_j$。
    
    当内层循环枚举到 $p_j = \text{最小质因数}(i)$ 时，触发了 `i % p_j == 0` 的条件，内层循环 `break`。
    
4. **结论**：
    
    这说明，当外层循环是 $i$ 时，它**最多只能**和 $\le \text{最小质因数}(i)$ 的质数相乘。
    
    回到我们的合数 $C = p_{min} \times i$。由于 $p_{min} \le \text{最小质因数}(i)$，这个组合必然会被枚举到。
    
    也就是说，合数 $C$ **一定**会在外层循环到达 $i$，且内层循环到达 $p_{min}$ 时被筛除。
    
    同时，因为 $C$ 的最小质因数 $p_{min}$ 是唯一的，$i = C / p_{min}$ 也是唯一的，所以**不会有第二种 $(i', p')$ 的组合能生成 $C$**。
    

**最终推论：**

既然每个合数 $C \le N$ 都只能被其唯一的 $(i, p_{min})$ 组合筛掉一次，那么内层语句 `is_prime[...] = false` 的总执行次数恰好等于 $\le N$ 的合数个数。

外层循环执行了 $N$ 次，内层循环总共执行了不超过 $N$ 次。

总时间复杂度 = 外层次数 + 内层总次数 = $O(N)$。

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
vector<int> primes;
bool isprime[100000005];
void build1(ll n)
{
	for (ll i = 1; i <= n; i++)
	{
		isprime[i] = true;
	}
	for (ll i = 2; i <= n; i++)
	{
		if (isprime[i])
		{
			primes.push_back(i);
			for (ll j = i * i; j <= n; j += i)
				isprime[j] = false;
		}
	}
}
void build2(ll n)
{
	isprime[0] = isprime[1] = true;
	for(ll i = 2; i <= n; i++)
	{
		if (!isprime[i])
			primes.push_back(i);
		for (ll j = 0; j < primes.size() && i * primes[j] <= n; j++)
		{
			isprime[i * primes[j]] = true;
			if (i % primes[j] == 0)
				break;
		}
	}
}
int main()
{
	ios::sync_with_stdio(false);
	cin.tie(0);
	int n, q, k;
	cin >> n >> q;
	build2(n);
	while (q--)
	{
		cin >> k;
		cout << primes[k - 1] << "\n";
	}
	return 0;
}
```
