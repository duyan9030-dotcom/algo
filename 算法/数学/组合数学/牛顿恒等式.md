**牛顿恒等式（Newton's Identities）**，又称为牛顿-吉拉德公式（Newton-Girard formulae），是代数学中的一个极其优美且实用的定理。

简单来说，它建立了一组变量的“幂和”**与**“初等对称多项式”之间的直接转换关系。

### 1. 两个核心概念：幂和 & 初等对称多项式

假设我们有 $n$ 个变量（或者说方程的 $n$ 个根）：$x_1, x_2, \dots, x_n$。

**概念一：幂和（Power Sums），记为 $P_k$**
****
顾名思义，就是每个变量的 $k$ 次方之和。

- $P_1 = x_1 + x_2 + \dots + x_n$
    
- $P_2 = x_1^2 + x_2^2 + \dots + x_n^2$
    
- $P_k = \sum_{i=1}^{n} x_i^k$


> **特点**：计算非常快！如果已知 $n$ 个数，算 $P_k$ 只需要把每个数求 $k$ 次方再相加，时间复杂度是 $O(n)$。

**概念二：初等对称多项式（Elementary Symmetric Polynomials），记为 $e_k$**

它代表的是从 $n$ 个变量中任选 $k$ 个变量相乘，然后把所有可能的组合加起来。

- $e_0 = 1$ （规定）
    
- $e_1 = x_1 + x_2 + \dots + x_n$ （任选1个，其实就是 $P_1$）
    
- $e_2 = \sum_{1 \le i < j \le n} x_i x_j$ （任选2个相乘再相加）
    
- $e_3 = \sum_{1 \le i < j < l \le n} x_i x_j x_l$ （任选3个相乘再相加，这就是你那道题里的三元组方案数）
    
- $e_n = x_1 x_2 \dots x_n$ （全选）

> **特点**：计算非常慢！如果暴力算 $e_k$，相当于从 $n$ 个数里挑 $k$ 个数的组合 $\binom{n}{k}$，当 $n$ 很大时，计算量是爆炸的 $O(n^k)$。

### 2. 牛顿恒等式的核心公式

牛顿恒等式的作用，就是**用好算的 $P_k$ 去推导难算的 $e_k$**（或者反过来）。

它的通用公式如下（对于 $k \le n$）：

$$k e_k = \sum_{i=1}^{k} (-1)^{i-1} e_{k-i} P_i$$

如果把求和符号展开，它实际上是一个递推式：

$$k e_k = e_{k-1}P_1 - e_{k-2}P_2 + e_{k-3}P_3 - \dots + (-1)^{k-1} e_0 P_k$$

### 3. 最常用的前几项展开（建议刻在脑子里）

在算法竞赛或日常推导中，我们极少直接套用那个带有 $\Sigma$ 的大公式，而是直接使用它展开后的前几项。

令 $k=1, 2, 3, 4$，代入公式并移项，我们可以得到**由幂和 $P$ 求解组合 $e$** 的直接公式：

- **当 $k=1$ 时：**
    
    $$e_1 = P_1$$
    
    _(选1个的和 = 1次方的和，废话但正确)_
    
- **当 $k=2$ 时：**
    
    $$2e_2 = e_1 P_1 - P_2$$
    
    变形得到：
    
    $$e_2 = \frac{P_1^2 - P_2}{2}$$
    
    _(这就是完全平方式 $(a+b+c)^2 = (a^2+b^2+c^2) + 2(ab+bc+ca)$ 的推广)_
    
- **当 $k=3$ 时：**（你在上一题中用到的）
    
    $$3e_3 = e_2 P_1 - e_1 P_2 + P_3$$
    
    变形得到：
    
    $$e_3 = \frac{e_2 P_1 - e_1 P_2 + P_3}{3}$$
    
- **当 $k=4$ 时：**
    
    $$4e_4 = e_3 P_1 - e_2 P_2 + e_1 P_3 - P_4$$
    
    变形得到：
    
    $$e_4 = \frac{e_3 P_1 - e_2 P_2 + e_1 P_3 - P_4}{4}$$
    

观察规律：符号始终是 $+ - + -$ 交替，且等号右边的下标满足组合：$e$ 的下标 + $P$ 的下标 $= k$。

### 4. 为什么它在算法题里是“神技”？

回到上一题的场景，假设你在树的某一个节点上，它有 $1000$ 个分支，你需要计算从这 $1000$ 个分支中**任选 $3$ 个不同分支**的大小乘积之和（即 $e_3$）。

- **如果不用牛顿恒等式（暴力组合）：**
    
    你需要写三个 `for` 循环嵌套，时间复杂度是 $\binom{1000}{3} \approx 1.6 \times 10^8$ 次运算，这仅仅是一个节点！对于 $10^5$ 级别的树，必定超时。
    
- **如果用牛顿恒等式：**
    
    你只需要遍历这 $1000$ 个分支**一次**：
    
    1. 计算 $P_1 = \sum S_i$
        
    2. 计算 $P_2 = \sum S_i^2$
        
    3. 计算 $P_3 = \sum S_i^3$
        
        一共只需 $1000$ 次循环（即 $O(n)$ 复杂度）。算完后，直接套用 $e_2$ 和 $e_3$ 的公式，**$O(1)$ 的时间就能得出原本需要 $O(n^3)$ 才能算出的结果**。
        

这就是牛顿恒等式在计算机科学中最核心的价值：**将高维度的组合数求和问题，降维成线性的幂和问题（降维打击）。**

### 5.推导

#### 1. 明确我们要联系的两个核心多项式

**基本对称多项式 (Elementary Symmetric Polynomials, 记为 $e_k$)：**

$$e_0 = 1$$

$$e_1 = x_1 + x_2 + \dots + x_n$$

$$e_2 = \sum_{1 \le i < j \le n} x_i x_j$$

$$e_k = \sum_{1 \le i_1 < i_2 < \dots < i_k \le n} x_{i_1} x_{i_2} \dots x_{i_k}$$

**等幂求和多项式 (Power Sums, 记为 $p_k$)：**

$$p_k = x_1^k + x_2^k + \dots + x_n^k$$

#### 2. 构造母函数并展开

我们引入一个辅助变量 $t$，构造一个根为 $\frac{1}{x_i}$ 的多项式（母函数）：

$$F(t) = \prod_{i=1}^n (1 - x_i t)$$

将这个连乘积完全展开，你会发现 $t^k$ 的系数刚好对应基本对称多项式 $e_k$：

$$F(t) = 1 - e_1 t + e_2 t^2 - e_3 t^3 + \dots + (-1)^n e_n t^n = \sum_{k=0}^n (-1)^k e_k t^k$$

#### 3. 对数求导变形（关键步骤）

为了把等幂和 $p_k$ 变出来，我们需要对连乘积进行“降维”变成加法。对 $F(t)$ 两边同时取自然对数：

$$\ln F(t) = \sum_{i=1}^n \ln(1 - x_i t)$$

接着，对 $t$ 进行求导：

$$\frac{F'(t)}{F(t)} = \sum_{i=1}^n \frac{-x_i}{1 - x_i t}$$

#### 4. 泰勒级数展开引入等幂和

利用无穷等比数列求和公式（即麦克劳林展开：$\frac{1}{1 - x} = 1 + x + x^2 + \dots$）：

$$\frac{-x_i}{1 - x_i t} = -x_i (1 + x_i t + x_i^2 t^2 + \dots) = -\sum_{j=1}^{\infty} x_i^j t^{j-1}$$

将这个展开式代入求导后的等式中，并交换求和的顺序：

$$\frac{F'(t)}{F(t)} = -\sum_{i=1}^n \sum_{j=1}^{\infty} x_i^j t^{j-1} = -\sum_{j=1}^{\infty} \left( \sum_{i=1}^n x_i^j \right) t^{j-1}$$

注意看括号里的部分，$\sum_{i=1}^n x_i^j$ 正是等幂和 $p_j$！于是我们得到：

$$\frac{F'(t)}{F(t)} = -\sum_{j=1}^{\infty} p_j t^{j-1}$$

为了方便对比系数，把分母乘到右边：

$$F'(t) = -F(t) \sum_{j=1}^{\infty} p_j t^{j-1}$$

#### 5. 比较系数得出最终恒等式

现在，我们将 $F(t)$ 和 $F'(t)$ 的多项式形式代入上式。

**左边 $F'(t)$ 的展开式：**

$$F'(t) = \frac{d}{dt} \sum_{k=0}^n (-1)^k e_k t^k = \sum_{k=1}^n (-1)^k k e_k t^{k-1}$$

**右边的展开式：**

$$- \left( \sum_{m=0}^n (-1)^m e_m t^m \right) \left( \sum_{j=1}^{\infty} p_j t^{j-1} \right)$$

为了得到牛顿恒等式，我们比较等式左右两边 $t^{k-1}$ 的系数。

- **左边 $t^{k-1}$ 的系数**，直接看出来是：$(-1)^k k e_k$
    
- **右边要凑出 $t^{k-1}$**，需要满足 $m + (j - 1) = k - 1$，即 $j = k - m$。因为 $j \ge 1$，所以 $m$ 最多取到 $k-1$。
    
    因此，右边 $t^{k-1}$ 的系数是：
    
    $$- \sum_{m=0}^{k-1} (-1)^m e_m p_{k-m}$$
    

令 $i = k - m$，当 $m$ 从 $0$ 遍历到 $k-1$ 时，$i$ 从 $k$ 遍历到 $1$。代换后的右边系数变为：

$$- \sum_{i=1}^k (-1)^{k-i} e_{k-i} p_i$$

**令左右两边系数相等：**

$$(-1)^k k e_k = - \sum_{i=1}^k (-1)^{k-i} e_{k-i} p_i$$

最后化简符号，等式两边同乘 $(-1)^{-k}$：

$$k e_k = - \sum_{i=1}^k (-1)^{-i} e_{k-i} p_i$$

由于 $-(-1)^{-i} = -(-1)^i = (-1)^{i-1}$，代入即可得到著名的**牛顿恒等式**：

$$k e_k = \sum_{i=1}^k (-1)^{i-1} e_{k-i} p_i$$
