佩尔方程（Pell's Equation）是数论中一类非常经典且优美的**二次丢番图方程**（Diophantine Equation）。在数学历史和算法竞赛中，它都占有核心地位，因为它将**连分数展开**、**矩阵快速幂**与**不定方程求解**完美地结合在一起。
## 一、 定义与标准形式

佩尔方程的标准形式为：

$$x^2 - ny^2 = 1$$

其求解前提和限制条件如下：

- **$n$ 为正整数且非完全平方数**：如果 $n$ 是完全平方数（例如 $n = k^2$），则方程可以因式分解为 $(x - ky)(x + ky) = 1$。在整数范围内，仅有平凡解 $(x, y) = (\pm 1, 0)$，毫无讨论价值。
    
- **寻找正整数解**：通常我们只讨论 $x > 0, y > 0$ 的正整数解。已知 $(1, 0)$ 永远是方程的一个平凡解。

## 二、 如何求解佩尔方程

求解佩尔方程的核心在于找到**最小正整数解**（也叫**基本解**，记作 $(x_1, y_1)$），随后所有的无限多组解都可以通过基本解快速推导出。

### 1. 暴力枚举法（适用于极小数据）

从 $y = 1, 2, 3, \dots$ 开始从小到大枚举 $y$，检查 $1 + ny^2$ 是否是一个完全平方数。

- **缺点**：佩尔方程的解增长极其剧烈！例如当 $n = 61$ 时，基本解已经高达 $x = 1766319049, y = 226153980$。暴力法会在某些极小数字上直接超时或溢出。

### 2. 连分数法（标准且最快解法）

任何非完全平方数 $\sqrt{n}$ 都可以展开为**循环简单连分数**：

$$\sqrt{n} = [a_0; \overline{a_1, a_2, \dots, a_m, 2a_0}]$$

其循环节长度记为 $m$。

我们可以逐步计算这个连分数的**渐进分数**（Convergents） $\frac{p_k}{q_k}$。根据佩尔方程定理，基本解 $(x_1, y_1)$ 必定出现在第一周期末尾附近的渐进分数中：

- 如果循环节长度 $m$ 为**偶数**，则基本解为第一周期倒数第一项渐进分数：$(x_1, y_1) = (p_{m-1}, q_{m-1})$。
    
- 如果循环节长度 $m$ 为**奇数**，则基本解为第二周期倒数第一项渐进分数：$(x_1, y_1) = (p_{2m-1}, q_{2m-1})$。

## 三、 核心数学性质与通解递推

只要找到基本解 $(x_1, y_1)$，佩尔方程具有以下极其优美的代数结构：

### 1. 解的无限性与代数共轭性质

由代数扩张域的范数乘法性质，若 $(x_1, y_1)$ 满足 $x_1^2 - ny_1^2 = 1$，则第 $k$ 组解 $(x_k, y_k)$ 可以通过幂次直接生成：

$$x_k + y_k\sqrt{n} = (x_1 + y_1\sqrt{n})^k \quad (k \in \mathbb{Z}^+)$$

### 2. 双向递推公式（矩阵计算的核心）

将上述公式展开或者对共轭式操作，可得到相邻解之间的线性递推关系：

- **一阶交叉递推：**
    
    $$x_{k+1} = x_1 x_k + n y_1 y_k$$
    
    $$y_{k+1} = y_1 x_k + x_1 y_k$$
    
- **二阶线性递推（避免交叉项）：**
    
    $$x_k = 2x_1 x_{k-1} - x_{k-2}$$
    
    $$y_k = 2x_1 y_{k-1} - y_{k-2}$$

### 3. 负佩尔方程

变体方程 $x^2 - ny^2 = -1$ **不一定有解**。它存在整数解充要条件是：$\sqrt{n}$ 连分数展开的**循环节长度 $m$ 为奇数**。

# 题解：

## 一、 题目大意与数学化归

给定正整数 $k = 114514$，求第 $k$ 小的正整数 $n$，使得 $2n + 1$ 和 $3n + 9$ 均为完全平方数，结果对 **223999** 取模。

### 1. 联立消元

设 $2n + 1 = a^2$，$3n + 9 = b^2$，其中 $a, b$ 为正整数。

由第一个方程可得 $n = \frac{a^2 - 1}{2}$，将其代入第二个方程：

$$3 \cdot \frac{a^2 - 1}{2} + 9 = b^2$$

两边同乘 2 消除分母：

$$3a^2 - 3 + 18 = 2b^2 \implies 2b^2 - 3a^2 = 15$$

### 2. 换元转化为标准形式

观察到 $2b^2 - 3a^2 = 15$ 中，右边 15 可以被 3 整除，且 $-3a^2$ 包含因子 3，因此 $2b^2$ 必须是 3 的倍数。由于 2 与 3 互质，$b$ 必然是 3 的倍数。

令 $b = 3c$（$c$ 为正整数），代入上式得：

$$2 \cdot (3c)^2 - 3a^2 = 15 \implies 18c^2 - 3a^2 = 15$$

两边同除以 3，整理得到一个**广义佩尔方程**：

$$a^2 - 6c^2 = -5$$

至此，问题转化为求解广义佩尔方程 $x^2 - 6y^2 = -5$ 的正整数解 $(x, y)$，其中 $x = a$ 且由于 $n = \frac{a^2 - 1}{2}$ 必须为正整数，需满足 **$x$ 为奇数**且 **$x > 1$**。

## 二、 广义佩尔方程与解族划分

### 1. 寻找基础解

首先考虑对应的齐次方程（标准佩尔方程）：

$$x^2 - 6y^2 = 1$$

显然其最小正整数基本解为 $(5, 2)$。

对于广义佩尔方程 $x^2 - 6y^2 = -5$，通过从小到大枚举可以找到两组**独立的基础初始解**：

- **初始解一：$(1, 1)$**。此时 $n = \frac{1^2-1}{2} = 0$，虽然不符合题目 $n \ge 1$ 的要求，但它是生成后续合法解的核心基底。
    
- **初始解二：$(7, 3)$**。此时 $n = \frac{7^2-1}{2} = 24$，这是所有合法正整数解中的第一小解。


### 2. 二阶线性递推关系

已知任一初始解 $(x_0, y_0)$ 乘以标准方程的解生成元 $(5 + 2\sqrt{6})$，即可得到下一组解：

$$x_{n+1} + y_{n+1}\sqrt{6} = (x_n + y_n\sqrt{6})(5 + 2\sqrt{6})$$

展开展开虚实部对比，得到一阶交叉递推公式：

$$\begin{cases} x_{n+1} = 5x_n + 12y_n \\ y_{n+1} = 2x_n + 5y_n \end{cases}$$

消去 $y$ 后，可得到仅关于 $x$ 的**二阶线性递推公式**：

$$x_{n+2} = 10x_{n+1} - x_n$$

## 三、 奇偶项交替规律与序列合并

分别从上述两组初始解出发，利用递推公式 $x_{m} = 10x_{m-1} - x_{m-2}$ 生成各自的 $x$ 序列：

- **解族 1（从 (7, 3) 开始递推）：** $7, 71, 703, 6959, \dots$
    
- **解族 2（从 (1, 1) 开始，下一项为 (17, 7)）：** $1, 17, 169, 1673, \dots$（排除非法项 $x=1$ 后为 $17, 169, 1673, \dots$）
    

将两个合法序列按数值从小到大合并，得到全集 $x$ 序列：

$$7, 17, 71, 169, 703, 1673, \dots$$

对应的必胜棋子数 $n$ 序列为：

$$24, 144, 2520, 14280, 247104, 1399464, \dots$$

| **序列项次 k**                   | **对应解族** | **该项 xk​ 的值**                     | **递推基底与规律**                                      |
| ---------------------------- | -------- | --------------------------------- | ------------------------------------------------ |
| **奇数项** ($k=1, 3, 5, \dots$) | 解族 1     | $a_1=7, a_3=71, a_5=703 \dots$    | 初始项 $x_0 = 7, x_1 = 71$，递推 $m = \frac{k-1}{2}$ 次 |
| **偶数项** ($k=2, 4, 6, \dots$) | 解族 2     | $a_2=17, a_4=169, a_6=1673 \dots$ | 初始项 $x_0 = 1, x_1 = 17$，递推 $m = \frac{k}{2}$ 次   |

## 四、 模逆元运算与复杂度分析

求出第 $k$ 个目标 $x$（记为 $a_k$）后，终极目标是计算 $n \pmod{223999}$：

$$n \equiv \frac{a_k^2 - 1}{2} \pmod{223999}$$

因为是在模意义下做除法，必须将除以 2 转化为**乘以 2 的模逆元**。由于模数 $223999$ 为质数，根据费马小定理或直接观察，2 的逆元为 $\frac{223999+1}{2} = 112000$，也可以通过快速幂 `qpow(2, mod - 2)` 求解。

- **时间复杂度**：本题采用线性递推，循环 $m \approx \frac{k}{2}$ 次。当 $k = 114514$ 时，循环次数约为 $57257$ 次，复杂度为 $O(k)$，在不到 2 毫秒内即可跑完，完全符合要求。如果 $k$ 大到 $10^{18}$，可进一步结合矩阵快速幂优化至 $O(\log k)$。

AC代码

```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
const ll mod = 223999;
// 快速幂求模逆元
ll qpow(ll x, ll y) {
    ll res = 1;
    x %= mod;
    while (y > 0) {
        if (y & 1) res = (res * x) % mod;
        x = (x * x) % mod;
        y >>= 1;
    }
    return res;
}
void solve() {
    ll k = 114514; // 题目给定的 k
    // 特例直接处理（可选防越界）
    if (k == 1) { cout << 24 << "\n"; return; }
    if (k == 2) { cout << 144 << "\n"; return; }
    if (k == 3) { cout << 2520 << "\n"; return; }
    ll m, a, b, c;
    // 根据奇偶项分配所属解族的递推次数与起点
    if (k % 2 != 0) {
        m = (k - 1) / 2;
        a = 7;
        b = 71;
    } else {
        m = k / 2;
        a = 1;
        b = 17;
    }
    // 线性递推求出目标 x 值
    for (ll i = 2; i <= m; i++) {
        c = (10 * b % mod - a + mod) % mod;
        a = b;
        b = c;
    }
    // 计算最终的必胜棋子数 n
    ll x = (m == 0 ? a : b); // 注意边界：若 m=1 直接取 b，大于 1 取递推结束后的 b
    ll ans = ((x * x % mod - 1 + mod) % mod * qpow(2, mod - 2)) % mod;
    cout << ans << "\n";
}
int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    solve();
    return 0;
}
```
