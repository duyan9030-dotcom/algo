### 1. 队列的核心概念 (The Concept)

队列是一种**线性数据结构**。你可以把它想象成在超市收银台前的**排队队伍**或者在银行等候叫号的人群。

它遵循一个最核心的原则：**先进先出**（**FIFO** - First-In-First-Out）。

- **入队 (Enqueue)**：新来的人只能排在队伍的**末尾**（队尾 / Rear）。
    
- **出队 (Dequeue)**：最先排队的人可以最先办理业务，并从队伍的**最前方**离开（队头 / Front）。

不允许有人插队，也不允许中间的人突然离开（在严格的经典队列定义中）。

### 2. 队列的重要性质 (Key Properties)

理解队列的性质有助于我们在合适的场景下选择使用它：

- **操作受限**：与普通的数组或链表不同，队列不允许在中间随意插入或删除元素。你只能在**一端插入（队尾），在另一端删除（队头）**。
    
- **时间复杂度**：在优秀的实现方式下，队列的入队（Enqueue）和出队（Dequeue）操作的时间复杂度都应该是 $O(1)$，即无论队列中有多少个元素，插入和删除都能在常数时间内完成。
    
- **保序性**：进入队列的元素会严格保持它们进入的先后顺序，直到它们离开。这在需要处理有序任务流时非常关键。

### 3. 队列的实现方式 (Implementation)

在编程中，队列通常是通过封装更底层的数据结构来实现的，主要有两种方式：

#### 方式 A：基于数组实现 (Array-based Queue)

- **原理**：分配一块连续的内存空间，用两个指针（或者索引）`front` 和 `rear` 分别指向队头和队尾。
    
- **挑战（假溢出问题）**：随着不断的入队和出队，所有的元素都会慢慢向数组的后方移动。即使数组前面有空位，`rear` 指针也可能到达了数组的末尾，导致无法继续插入。
    
- **解决方案（循环队列 Circular Queue）**：将数组在逻辑上首尾相连。当 `rear` 到达数组末尾时，如果数组开头还有空位，就让 `rear` 绕回到数组的开头。

#### 方式 B：基于链表实现 (Linked-list-based Queue)

- **原理**：使用单向链表。链表的头部作为队头（Front），链表的尾部作为队尾（Rear）。
    
- **优势**：不需要预先分配固定大小的内存空间，理论上可以无限增长（只受限于系统内存）。
    
- **操作**：
    
    - **入队**：在链表尾部追加一个新节点，并更新 `rear` 指针。
        
    - **出队**：移除链表的第一个节点，并更新 `front` 指针指向下一个节点。
[队列模板](https://www.luogu.com.cn/problem/B3616)
纯C语言实现：
```cpp
#include <bits/stdc++.h>
using namespace std;
int que[10005];
int front = 0, tail = 0;
void push(int x)
{
    que[tail++] = x;
}
void pop()
{
    front++;
}
bool exits_top()
{
    return front != tail;
}
int query()
{
    return que[front];
}
int size()
{
    return tail - front;
}
int main()
{
    int n;
    cin >> n;
    int op,x;
    while (n--)
    {
        cin >> op;
        if (op==1)
        {
            cin >> x;
            push(x);
        }
        else if (op==2)
        {
            if (exits_top())
            {
                pop();
            }
            else
            {
                cout << "ERR_CANNOT_POP" << "\n";
            }
        }
        else if (op == 3)
        {
            if (exits_top())
            {
                cout << query() << "\n";
            }
            else
            {
                cout << "ERR_CANNOT_QUERY" << "\n";
            }
        }
        else if (op == 4)
        {
            cout << size() << "\n";
        }
    }
}
```

C++中的STL实现：
```cpp
#include <bits/stdc++.h>
using namespace std;
int que[10005];
int front = 0, tail = 0; 
int main()
{
    int n;
    cin >> n;
    int op,x;
    queue<int> q;
    while (n--)
    {
        cin >> op;
        if (op==1)
        {
            cin >> x;
            q.push(x);
        }
        else if (op==2)
        {
            if (!q.empty())
            {
                q.pop();
            }
            else
            {
                cout << "ERR_CANNOT_POP" << "\n";
            }
        }
        else if (op == 3)
        {
            if (!q.empty())
            {
                cout << q.front() << "\n";
            }
            else
            {
                cout << "ERR_CANNOT_QUERY" << "\n";
            }
        }
        else if (op == 4)
        {
            cout << q.size() << "\n";
        }
    }
}
```
