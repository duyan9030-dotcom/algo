前置知识：[[DFS]][[链式前向星]]
定义 $p(u, v)$ 为 $u$ 到 $v$ 路径上所有节点值的乘积。如果一个三元组 $\{u, v, w\}$ 是“好的 (good)”，则必须满足： $P = p(u, v) \cdot p(v, w) \cdot p(w, u)$ 是一个完全平方数。求树中所有好的三元组数量。

在树结构中，任意三个不同的节点 $u, v, w$ 都会有一个唯一的“中心交点”（也叫中位点，Median），设这个点为 $c$。这三个点到 $c$ 的路径形成了三个分支（如果某个点本身就是交点，则对应分支长度为0）。
我们来统计这三条路径 $u \to v$，$v \to w$，$w \to u$ 中，各个节点被包含的次数：

- $u \to v$ 的乘积 = $P(u, c) \cdot P(v, c) \cdot a_c$
    
- $v \to w$ 的乘积 = $P(v, c) \cdot P(w, c) \cdot a_c$
    
- $w \to u$ 的乘积 = $P(w, c) \cdot P(u, c) \cdot a_c$

三条路径的权值乘起来就是：

$$
P=a_c^3 \cdot P(u, c)^2 \cdot P(v, c)^2 \cdot P(w, c)^2
$$

我们可以将乘积 $P$ 写成：

$$
P = a_c^3 \cdot \prod_{x \neq c} a_x^2 = a_c \cdot \left( a_c \cdot \prod_{x \neq c} a_x \right)^2
$$

**结论**：因为后面是一个完全平方数，所以乘积 $P$ 是完全平方数 **当且仅当交点 $a_c$ 本身是一个完全平方数**。

### 2. 算法策略
基于上述结论，问题转化为：

3. 枚举树上的每一个节点 $i$。
    
4. 检查 $a_i$ 是否为完全平方数。如果是，我们需要计算以 $i$ 为交点的三元组 $\{u, v, w\}$ 的数量。

**如何计算以 $i$ 为交点的三元组数量？**

如果我们在树中把节点 $i$ 删掉，整棵树会分裂成若干个连通块（子树）。

一个三元组 $\{u, v, w\}$ 想要以 $i$ 为交点，这三个节点必须分布在 **不同的** 连通块中。

特别地，节点 $i$ 自己也可以作为三元组中的一个点，所以我们可以把节点 $i$ 自身看作是一个大小为 $1$ 的独立连通块。

假设节点 $i$ 切割出来的各个连通块大小分别为 $s_1, s_2, \dots, s_k$，以及它自己 $s_0 = 1$。

我们需要从这 $k+1$ 个数字中选出 $3$ 个，求所有可能的乘积之和。即求：
$$e_3 = \sum_{x < y < z} s_x s_y s_z$$
为了在 $O(\text{度数})$ 的时间内快速算出这个结果，我们可以利用[[牛顿恒等式]] ：

令：

- $p_1 = \sum s_i = n$ （所有连通块大小之和就是节点总数）
    
- $p_2 = \sum s_i^2$
    
- $p_3 = \sum s_i^3$

那么有：

- $e_1 = p_1$
    
- $e_2 = \frac{e_1 p_1 - p_2}{2}$
    
- $e_3 = \frac{e_2 p_1 - e_1 p_2 + p_3}{3}$
### 3. 实现步骤

1. **建树与预处理**：随便选一个根（如节点 $1$），用一次 DFS 算出每个节点的子树大小 `sz[u]`，并记录它的父节点。
    
2. **枚举交点**：遍历所有节点 $i \in [1, n]$：
    
    - 用 `sqrt` 检查 $a_i$ 是否为完全平方数。
        
    - 如果不是，跳过。
        
    - 如果是，找出以 $i$ 为中心的所有连通块大小：
        
        - 对 $i$ 的每个子节点 $v$，大小为 `sz[v]`。
            
        - 如果 $i$ 有父节点，父节点所在那一坨的大小为 $n - sz[i]$。
            
        - 节点 $i$ 自己，大小为 $1$。
            
    - 用前面的 $O(k)$ 公式计算出 $e_3$ 并累加到答案中。

AC代码
```cpp
#include <bits/stdc++.h>
using namespace std;
#define ll long long
ll n;
ll a[200005];
ll head[400005];
ll nxt[400005];
ll to[400005];
ll sz[200005];
ll father[200005];
ll cnt = 1;
void addedge(ll u, ll v)
{
    nxt[cnt] = head[u];
    to[cnt] = v;
    head[u] = cnt++;
}
void dfs(ll u, ll fa)
{
    sz[u] = 1;
    father[u] = fa;
    for (ll ei = head[u]; ei > 0; ei = nxt[ei])
    {
        if (to[ei] != fa)
        {
            dfs(to[ei], u);
            sz[u] += sz[to[ei]];
        }
    }
}
bool issquare(ll i)
{
    ll r = sqrt(i);
    return r * r == i;
}
void solve()
{
    cin >> n;
    for (int i = 1; i <= n; i++)
    {
        cin >> a[i];
        head[i] = 0;
    }
    cnt = 1;
    for (int i = 1, u, v; i < n; i++)
    {
        cin >> u >> v;
        addedge(u, v);
        addedge(v, u);
    }
    dfs(1, 0);
    ll ans = 0;
    for (int i = 1; i <= n; i++)
    {
        if (issquare(a[i]))
        {
            ll p1 = n;
            ll p2 = 1;
            ll p3 = 1;
            for (int ei = head[i]; ei > 0; ei = nxt[ei])
            {
                ll s = 0;
                if (to[ei] == father[i])
                    s = n - sz[i];
                else
                    s = sz[to[ei]];
                p2 += s * s;
                p3 += s * s * s;
            }
            ll e1 = p1;
            ll e2 = (e1 * p1 - p2) / 2;
            ll e3 = (e2 * p1 - e1 * p2 + p3) / 3;
            ans += e3;
        }
    }
    cout << ans << '\n';
}
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int t;
    cin >> t;
    while (t--)
        solve();
}
```
