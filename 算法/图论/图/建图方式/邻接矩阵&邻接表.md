前置知识：[[图论基础]]

### 在图论算法中，建图（图的存储）是最基础也是最核心的一步。图由顶点（Vertex）和边（Edge）组成，我们通常用 $V$ 表示顶点数，用 $E$ 表示边数。

在 C++ 中，借助 STL（标准模板库），我们可以非常优雅地实现两种最主流的建图方式：**邻接矩阵（Adjacency Matrix）** 与 **邻接表（Adjacency List）**。

### 1. 邻接矩阵（Adjacency Matrix）

#### 核心原理

用一个二维数组 `adj[u][v]` 来记录从顶点 $u$ 到顶点 $v$ 的边：

如果是无权图通常初始化`adj[i][j]=0`代表没有边，`adj[i][j]==1`代表有边。

如果是有权图通常初始化`adj[i][j]==INF`为一个特殊值代表没有边，`adj[i][j]!=INF`代表有边，且权值为`adj[i][j]`。

```cpp
#include <iostream>
#include <vector>
using namespace std;

const int INF = 1e9; // 代表无穷大（无边）

int main() {
    int n = 5; // 顶点数 (假设编号从 1 到 n)
    int m = 4; // 边数
    
    // 使用二维 vector 初始化邻接矩阵，大小 (n+1) x (n+1)，初始值为 INF
    vector<vector<int>> adj(n + 1, vector<int>(n + 1, INF));
    
    // 自己到自己的距离通常设为 0
    for(int i = 1; i <= n; i++) {
        adj[i][i] = 0;
    }

    // 假设输入边的格式为：u v weight
    vector<vector<int>> edges = {
        {1, 2, 5},
        {1, 3, 2},
        {2, 4, 1},
        {3, 4, 7}
    };

    // 建图
    for(const auto& edge : edges) {
        int u = edge[0];
        int v = edge[1];
        int w = edge[2];
        
        adj[u][v] = w; 
        adj[v][u] = w; // 如果是有向图，去掉这一行即可
    }

    // 查询节点 1 到节点 2 的权重
    cout << "1 到 2 的边权: " << adj[1][2] << endl;

    return 0;
}
```

### 2. 邻接表（Adjacency List）

#### 核心原理

只保存“真正存在的边”。为每一个顶点 $u$ 维护一个动态数组（或者链表），存储所有从 $u$ 出发的边。

- 我们可以用 `std::vector` 的嵌套：`vector<vector<Edge>>`。
    
- 在稀疏图中，邻接表能极大节省内存并提升遍历效率。

#### C++ STL 实现代码

C++

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 定义边的结构体
struct Edge {
    int to;     // 目标顶点
    int weight; // 边权
};

int main() {
    int n = 5; // 顶点数
    
    // adj[u] 存储了所有从顶点 u 出发的边
    vector<vector<Edge>> adj(n + 1);

    // 假设输入边的格式：u v weight
    vector<vector<int>> edges = {
        {1, 2, 5},
        {1, 3, 2},
        {2, 4, 1},
        {3, 4, 7}
    };

    // 建图
    for(const auto& e : edges) {
        int u = e[0], v = e[1], w = e[2];
        
        adj[u].push_back({v, w});
        adj[v].push_back({u, w}); // 如果是有向图，去掉这一行
    }

    // 遍历顶点 1 的所有邻居
    cout << "顶点 1 的所有邻居及边权:" << endl;
    for(const auto& edge : adj[1]) {
        cout << " -> " << edge.to << " (权重: " << edge.weight << ")" << endl;
    }

    return 0;
}
```

### 3. 优缺点与应用场景对比

|**维度**|**邻接矩阵 (Adjacency Matrix)**|**邻接表 (Adjacency List)**|
|---|---|---|
|**空间复杂度**|$O(V^2)$|$O(V + E)$|
|**查询 $(u, v)$ 是否存在**|**极快**：$O(1)$|**较慢**：$O(\text{degree}(u))$，需遍历 $u$ 的链表|
|**遍历点 $u$ 的所有邻居**|**较慢**：$O(V)$，需扫描整行|**极快**：$O(\text{degree}(u))$，只遍历实际存在的边|
|**添加边**|$O(1)$|$O(1)$ (使用 `vector::push_back`)|
|**处理重边**|只能保存一条（常保留最小/最大权）|天然支持保存多条完全相同的边|
|**适用图类型**|**稠密图** ($E \approx V^2$)|**稀疏图** ($E \ll V^2$)|
|**适用规模极限**|顶点数 $V \le 3000$ 左右（否则内存超限）|顶点数 $V \le 10^6$ 甚至更大|
